Generating Random Number

Random numbers are numbers of a particular set or interval but these are unpredictable. Random number generation can be compared with Ludu Dice throwing. Before throwing the dice you can't predict what number will be displayed on the dice. Let's take a look how random number can be generated in a program. To generate random number rand() function is used. A code is given below for better understanding.
#include <iostream>
#include
using namespace std;
int main()
{
int randomNumber1 = rand();
int randomNumber2 = rand() % 2;
int randomNumber3 = rand() % 6+5;
cout << randomNumber1 << endl;
cout << randomNumber2 << endl;
cout << randomNumber3 << endl;
return 0;
}

****How rand() function works:****


In this code,randomNumber1, randomNumber2, randomNumber3 are three integer variables which generate three random number of three different ranges. randomNumber1 will generate any integer number. What about randomNumber2? . The number which will be written after the modulus operator(%) will indicate the range. We know integer numbers start from 0. rand()%2 means, it will generate a random number which is less than 2. That means that will generate 0 or 1. You may have question , how to create a random number within more specific range? You will find your answer by understanding the generating process of randomNumber3. Suppose, you want to generate a random number of range 5 to 10. The number should be equal to five or greater than five but less than or equal to ten. The mechanism is at first if you can find out the random number number of range 0 to 5, then if you add 5 with it then you will be able to generate a random number of range 5 to 10 by your program. According to this, in this program "rand()%6" has been written to generate a random number of range 0 to 5 .Then 5 has been added with this random number to find out the random number of range 5 to 10.
