Generating Random Number

Random numbers are numbers of a particular set or interval but these are unpredictable. Random number generation can be compared with Ludu Dice throwing. Before throwing the dice you can't predict what number will be displayed on the dice. Let's take a look how random number can be generated in a program. To generate random number rand() function is used. A code is given below for better understanding.

#include <iostream>
#include <cstdlib>
using namespace std;
int main()
{
int randomNumber1 = rand();
int randomNumber2 = rand() % 2;
int randomNumber3 = rand() % 6+5;
cout << randomNumber1 << endl;
cout << randomNumber2 << endl;
cout << randomNumber3 << endl;
return 0;
}


How rand() function works:


In this code,randomNumber1, randomNumber2, randomNumber3 are three integer variables which generate three random number of three different ranges. randomNumber1 will generate any integer number. What about randomNumber2? . The number which will be written after the modulus operator(%) will indicate the range. We know integer numbers start from 0. rand()%2 means, it will generate a random number which is less than 2. That means that will generate 0 or 1. You may have question , how to create a random number within more specific range? You will find your answer by understanding the generating process of randomNumber3. Suppose, you want to generate a random number of range 5 to 10. The number should be equal to five or greater than five but less than or equal to ten. The mechanism is at first if you can find out the random number number of range 0 to 5, then if you add 5 with it then you will be able to generate a random number of range 5 to 10 by your program. According to this, in this program "rand()%6" has been written to generate a random number of range 0 to 5 .Then 5 has been added with this random number to find out the random number of range 5 to 10. Linear Search

মনে করুন, আপনার কাছে কিছু চিরকুট আছে যেগুলো সরলরৈখিকভাবে সাজানো। প্রত্যেকটি চিরকুটে একটা করে সংখ্যা লিখা। আপনাকে বলা হল যেকোনো একটি সংখ্যা খুজে বের করার জন্য। আপনি একটা একটা করে খুজা শুরু করলেন। যদি পেয়ে যান, তাহলে আপনাকে বলতে হবে চিরকুটটা লাইনের কত নম্বরে ছিল। আর না পেলে বলবেন নেই। ব্যাপারটা Linear Search এর মত। মনে করেন, কোনো একটি অ্যারেতে অনেকগুলো ডাটা রয়েছে। অ্যারেতে ডাটা সরলরৈখিকভাবে থাকে। অ্যারের নির্দিষ্ট একটি ডাটা খুজে বের করার জন্য বিভিন্ন প্রকার সার্চিং প্রসেস এর মধ্যে একটি হল Linear Search. উদাহরণস্বরূপ, একটি অ্যারে নেওয়া যাক। arr[10]={34,55,77,98,30,37,12,15,78,3}; এখন যদি বলা হয় 98 ডাটাটি খুজে বের করে এর ইনডেক্স বের করার জন্য তাহলে অ্যারের প্রথম ইনডেক্স থেকে সার্চিং শুরু হবে। একটা একটা করে চেক করে 98 ডাটা পেয়ে তার ইনডেক্স প্রিন্ট করে দিলেই কাজ শেষ। আর এরকম কোনো ডাটা যদি সার্চ করার জন্য বলা হয় যেটি কিনা অ্যারেতে নেই তবে ইনডেক্স হিসেবে -1 আসবে। কারণ -1 কোনো অ্যারের ইনডেক্স হতে পারে না। এবার তাহলে কোড করা যাক:-

#include <iostream>
using namespace std;
int linearSearch(int data, int arr[], int sizeOfArray )
{
for (int i = 0; i <sizeOfArray ; i++)
{
if (arr[i] == data)
{
return i;
}
}
return -1;
}
int main()
{
int arr[10] = { 34,55,77,98,30,37,12,15,78,3 };
int sizeOfArray = 10;
int data;
cout << "Enter the data which you want to search: ";
cin >> data;
int index;
index=linearSearch(data, arr, sizeOfArray);
if (index == -1)
{ cout << "Ooops! Inputted data has not been found." << endl;
}
else
{ cout << "Inputted data has been found at " << index << " no index" << endl;
}
return 0;
}

এই কোডে Linear searching- এর কাজটি করছে linearSearch(int data, int arr[], int sizeOfArray) ফাংশনটি যার প্যারামিটার হিসেবে আছে int data (যে ডাটাটি আপনি সার্চ করছেন), int arr[]( অ্যারের এড্রেস), int sizeOfArray (অ্যারের সাইজ)। এই ফাংশনে একটি for লুপ ব্যবহার করে একটা একটা করে ডাটা এমনভাবে চেক করা হয়েছে যেন অ্যারের কোনো উপাদানের সাথে আপনার ইনপুট করা ডাটা মিলে গেলেই ওই ইনডেক্স ফাংশনে রিটার্ণ হবে। আর না মিললে for লুপ থেকে বেরিয়ে এসে -1 রিটার্ণ করে দিবে। পরে মেইন ফাংশনে রিটার্ণ ভ্যালু index ভ্যারিয়েবলে স্টোর হবে। তারপর চেক করা হবে এর ভ্যালু -1 কিনা? যদি -1 হয় তার মানে অ্যারেটিতে ইনপুট করা ডাটাটি নেই। আর যদি -1 না হয় তার মানে অ্যারেতে ডাটাটি রয়েছে। তারপর ওই ডাটা ইনডেক্স প্রিন্ট করা যাবে। আশা করি বুঝতে পেরেছেন। কোথাও বুঝতে সমস্যা হলে কমেন্ট করতে পারেন। ধন্যবাদ

Fractional Knapsack

এলাকার লাভার বয় রোমিও হঠাৎ অবগত হইল যে, তাহার দীর্ঘদিনের ক্রাশের পিতার দোকানে চুরি হইয়াছে। এই কথা শুনিয়া রোমিও হিরোর মত দৌড় মারিয়া ঘটনাস্থলে উপস্থিত হইল। ঘটনাস্থল পরিদর্শন করিয়া রোমিও জ্ঞানীর স্বরে বলিয়া উঠিল, " চোর ত দেখি ফ্র্যাকশনাল ন্যাপস্যাক অ্যালগরিদম অনুসরণ করিয়া চুরি করিয়াছে।" আশেপাশের সবাই ত থ বনিয়া গেল। রোমিও একজন প্রোগ্রামার। যাই হোক, রোমিওর কথা শুনিয়া তাহার ক্রাশের কৌতুহলী ছোটভাই জানিতে চাহিল, "ফ্র্যাকশনাল ন্যাপসেক? এ আবার কি জিনিস!" তারপর রোমিও ক্রাশের ছোটাভাইকে বুঝানো শুরু করল। তাহলে এখন দেখা যাক, রোমিও কিভাবে ফ্র্যাকশনাল ন্যাপসেক বুঝিয়েছিল:-
রোমিওর ক্রাশের বাবার দোকানে ছিল:-
১. ৫ কেজি চাউল - ৩০০ টাকা;
২. ২ কেজি পেয়াজ - ১২০ টাকা;
৩. ১ কেজি রসুন - ১০০ টাকা;
৪. ৩ কেজি মসুর ডাল- ৩৬০ টাকা।
৫. ৪ কেজি চিনি - ২৮০ টাকা।
৬. ৪ কেজি তৈল - ৫০০ টাকা।
মনে কর, চোরের কাছে একটি ছালা ছিল যেটাতে ১০ কেজি ওজনের বেশি কিছু নিলে ঐ ছালা ছিড়ে যাবে। চোরের লক্ষ্য ১০ কেজি পণ্য এমনভাবে নেয়া যাতে সর্বোচ্চ লাভ হয়। এজন্য চোর যেটা করেছিল, প্রথমে সবগুলো পণ্যের প্রতি কেজির মূল্য হিসাব করেছিল।তারপর যেগুলোর মূল্য বেশি সেগুলো আগে নিয়ে তার ১০ কেজি ছালা পূরণ করেছিল।
এখানে প্রতি কেজি হিসাব করে মূল্যের উর্ধ্বক্রমানুসারে সাজালে যা হয়:-
১ কেজি তৈল :- ১২৫ টাকা;
১ কেজি মসুর ডাল:- ১২০ টাকা;
১ কেজি রসুন:- ১০০ টাকা;
১ কেজি চিনি:- ৭০ টাকা;
১ কেজি পেয়াজ:- ৬০ টাকা;
১ কেজি চিনি:- ৫০ টাকা;
চোর যা করবে তা হল উপরের ক্রমানুসারে পণ্য চুরি করা শুরু করবে এবং প্রত্যেকবার চেক করবে মোট ওজন ১০ কেজি হয় কিনা? যখন ১০ কেজি হয়ে যাবে তখন আর নিবে না। আর যে ১০ কেজি ওজনের পণ্য ছালাতে থাকবে সেগুলো থেকে প্রাপ্ত লাভ সর্বোচ্চ হবে। তাহলে উপরিউক্ত পণ্যের জন্য সর্বোচ্চ লাভ হিসেব করা যাক:- তৈলের দাম যেহেতু বেশি সেহেতু তৈল আগে নেয়া শুরু করবে। মোট তৈল আছে ৪ কেজি যা ১০ কেজি থেকে কম। এ থেকে প্রাপ্ত লাভ:- ৫০০ টাকা। তারপর মসুর ডাল আছে ৩ কেজি। এখন ৩ কেজি নেয়ার পর মোট ওজন ৭ কেজি (৭ কেজি<১০ কেজি)। এ পর্যন্ত লাভ ( ৫০০+৩৬০) টাকা= ৮৬০ টাকা। তারপর ১ কেজি রসুন (৭+১ <১০)। লাভ (৮৬০+১০০) টাকা= ৯৬০ টাকা। তারপর ৪ কেজি চিনির পুরোটা নিতে পারবে না। কারণ (৮+৪>১০)। তারমানে ৪ কেজি ওজনের চিনির পুরোটা নিতে গেলে ওজন ১২ কেজি হয়ে যাবে যা ১০ কেজি থেকে বেশি। এজন্য ঐ ৪ কেজি এর ২ কেজি নেয়া যাবে। তাহলে ওজন সমান ১০ কেজি হবে। ফলে লাভ হবে:- (৯৬০+(৭০*২)) টাকা= ১১০০ টাকা। সুতরাং উপরের পণ্যের জন্য ১০ কেজির ক্ষেত্রে সর্বোচ্চ লাভ হবে ১১০০ টাকা। এবার মূল কাহিনীতে ফিরে আসা যাক:-
রোমিও তাহার ক্রাশের ছোট ভাইকে ফ্র্যাকশনাল ন্যাপসেক বুঝাতে সফল হইল। তাহা দেখিয়া রোমিওর ক্রাশের বাপজান মুগ্ধ হইয়া রোমিওর সাথে তাহার মেয়ের বিবাহের ঘোষণা করিল। অত:পর রোমিও বিবাহ করিয়া সুখে সংসার করিতে লাগিল।
এবার ফ্র্যাকশনাল ন্যাপসেক এর কোডটা দেখা যাক:-

#include <iostream>
using namespace std;
class product
{
public:
float weight;
float price;
float avgPrice;
};
int main()
{
int numberOfProducts;
cout<<"Enter the number of products: ";
cin >> numberOfProducts;
cout<<"Enter the maximum weight limits";
float maxWeight;
cin >> maxWeight;
product* obj= new product [numberOfProducts];
for (int i=0; i < numberOfProducts; i++)
{
cin >> obj[i].price;
cin >> obj[i].weight;
obj[i].avgPrice=obj[i].price/obj[i].weight;
}
for (int i=0; i< numberOfProducts; i++)
{
for (int j=i+1; j < numberOfProducts; j++)
{
if (obj[i].avgPrice < obj[j].avgPrice)
{
product temp;
temp=obj[i];
obj[i]=obj[j];
obj[j]=temp;
}
}
}
float price=0;
for (int i=0; i < numberOfProducts; i++)
{
if(obj[i].weight==maxWeight)
{
price=price+obj[i].price;
break;
}
if (obj[i].weight << maxWeight)
{
price=price+obj[i].price;
maxWeight=maxWeight-obj[i].weight;
continue;
}
if (obj[i].weight>maxWeight)
{
price=price+maxWeight*obj[i].avgPrice;
break;
}
}
cout<<"Total price:"<< price << endl;
return 0;
}
